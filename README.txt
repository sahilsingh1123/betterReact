# Installing react related modules
## For creating react-app directly with create-react-app command

1. sudo apt install npm (make sure node version is above 14)
2. sudo npm install create-react-app (
    this will download the create-react-app utility locally
)
2. sudo npx create-react-app test-app
3. create-react-app test-app
    - if failed with version issue
    - install nvm if not installed
        - curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
    - nvm install 14
    - nvm use 14
